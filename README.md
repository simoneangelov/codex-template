# Codex Template

A project-agnostic autonomous development system for [OpenAI Codex](https://openai.com/codex) and [Claude Code](https://claude.ai/code).

Drop `.codex/` into any repository to give your AI agent an opinionated, Linear-driven workflow for planning, implementing, reviewing, and completing issues.

## Getting started

See [`.codex/SETUP.md`](.codex/SETUP.md) for setup instructions.

## What's included

**Briefs** — role-scoped instructions for specialized agent jobs (issue analysis, repo exploration, solution planning, testing, review prep)

**Workflows** — end-to-end commands that orchestrate the full development lifecycle (`/plan-issue`, `/implement-issue`, `/revise-plan`, `/complete-issue`, `/create-issue`, `/discuss-issue`)
